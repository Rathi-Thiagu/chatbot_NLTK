Resource: Google colab 
Architecture of Chatbots : 
chat window or session -- Interface --- NLP model ---corpus or Application DB 
How does a chatbot work ?
    Import corpus ---needed for training the model  
    Preprocess the data --- data cleaning
    Text case handling ---convert all the data coming as input to either upper case or lower 
    Tokenization --- This is a blog 
    Stemming --- finding similarities between words with same root word 
    Bag of Words (BOW) -- process of converting words into numbers by generating vector embeddings from the tokens generated 
    One hot encoding
